# Idealogue Static Boilerplate

Static HTML boilerplate running on Node.js

- assemble.io
- handlebars template (.hbs)
- sasslang css (.scss)
- simple js setup
- grunticon

## Development

How to set up your local development environment, after cloning the repository.

1. `npm install`
2. `grunt`
3. `grunt watch`
4. `node server`
